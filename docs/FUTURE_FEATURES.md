# Future Features (Post-Hackathon)

These are explicitly out-of-scope for the current hackathon demo build.

## Current Baseline (for context)

- Dashboard backend is MongoDB-based.
- Mobile app is offline-first with SQLite.
- Cloud sync currently runs one-way: mobile to dashboard.
- Supabase is optional and not the primary data backend.

## 1) Data and Sync Evolution

- Two-way sync between dashboard and mobile.
- Conflict resolution strategy for offline edits from multiple devices.
- Selective sync rules (by date/account/category) for faster sync windows.

## 2) AI and Intelligence Layer

- Better categorization using ML-assisted mapping on top of current parser system.
- Family-level advisory assistant with grounded context and safe response controls.
- Forecast and anomaly insights with clearer explanation output.

## 3) Product and Collaboration

- Stronger family workflows (goal collaboration, approval actions, accountability views).
- Better onboarding for family invite and role management.
- More analytics depth for category, member, and trend interpretation.

## 4) Compliance and Operations

- Enhanced export packs and reporting automation.
- Complete audit coverage for all critical actions.
- Production-grade observability and alerting.
